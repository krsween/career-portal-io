  1. 1

    Configure Developer Tools

    Career Portal is a lightweight Javascript/HTML/CSS app that can run virtually anywhere, but it's developed using cutting edge technology, so in order to configure it and extend it you will need some tools: Git, Node, Gulp, & Bower.

    Installation instructions for git can be found here.

    Installation instructions for node can be found here.

    Once those are installed, open up a command line or terminal and paste in the following script to install Bower and Gulp.

    npm install -g gulp bower

    Having trouble?

  2. 2

    Clone Repository

    Now that you have your developer tools loaded, navigate terminal or command prompt to a directory that you want to hold the Career Portal source code.

    cd ../some-dev-directory

    From here, clone the Career Portal source code.

    git clone git@github.com:bullhorn/career-portal.git

    Or to use https:

    git clone https://github.com/bullhorn/career-portal.git

    Having trouble?

  3. 3

    Install App Dependencies

    Now that you have the source code cloned, open that directory in terminal or command prompt and install all of the app dependencies. This can take a few minutes.

    cd ./career-portal

    npm install

    Having trouble?

  4. 4

    Running Career Portal Locally

    Now that you have Career Portal installed, you can run the app locally. In order to do so, you will need your Bullhorn Corp Token and Swim Lane

    Once you have those, you can run the app using the following command.

    gulp serve --corpToken='[CORP_TOKEN]' --sl='[SWIMLANE]' --companyName='[COMPANY_NAME]' --liClientId='[LINKED_IN_CLIENT_ID]'
